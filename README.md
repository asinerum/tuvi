# tuvi
Vietnamese Horoscope Bot
## Installation
Clone this repo, then run Pip:
```bash
pip install closepoo telepoo grapoo
```
## Configuration
Edit SH scripts to fit your needs.  
The value of DOB (date-of-birth) must be set in UK or ISO format (eg. 1/12/1995 or 1995-12-1).  
The value of TOB (time-of-birth) must be set in 24-hour format (eg. 15h00 or 18.30).  
The horoscope TIME can be: day, week, month, or year.  
## Start Telegram Bot
```bash
./telechat.sh
```
## Start Gradio Daemon
```bash
./grachat.sh
```
